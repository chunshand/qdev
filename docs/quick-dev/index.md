## 开发调试


```shell

# 我用的是yarn 其他把管理器随意 大同小异

# 启动服务
cd ./serve
yarn run dev

# 启动后台
cd ./admin
yarn run dev

# 启动文档
cd ./docs
yarn run docs:dev
```
